# Руководство пользователя приложения «Моделирование для медицинских целей»

Это приложение предназначено для анализа медицинских данных, построения моделей машинного обучения и получения интерпретируемых прогнозов. Приложение сочетает в себе визуальный интерфейс, автоматический разбор структуры данных и интегрированные инструменты моделирования, что позволяет пользователю выполнять полный цикл анализа — от загрузки файла до получения готовой модели и объяснений.

---

## Загрузка данных
В начале работы необходимо загрузить файл в формате **CSV** или **Excel**.  
После загрузки приложение автоматически отображает базовую информацию:

- количество строк и столбцов;
- типы признаков (числовые, категориальные);
- наличие пропусков;
- предварительная структура таблицы.

Также доступна **описательная статистика**, включающая основные метрики: среднее значение, медиану, стандартное отклонение, минимумы и максимумы. Это позволяет оценить качество данных перед переходом к моделированию.

---

## Подключение ИИ
После того как данные загружены, становится доступна кнопка **«Подключить ИИ»**.  

При активации ИИ приложение автоматически анализирует:

- структуру признаков;
- распределение целевой переменной (если она присутствует);
- особенности данных, влияющие на выбор модели.

Этот шаг подготавливает внутренние параметры и позволяет корректно использовать инструменты машинного обучения без обязательной ручной предварительной обработки.

После подключения ИИ становятся доступны разделы моделирования.

---

## Логистическая регрессия
Раздел логистической регрессии предназначен для построения простой и понятной модели классификации, которая особенно подходит для медицинских задач, где важны интерпретируемость и прозрачность.

### Возможности раздела
Пользователь может:

- выбрать или уточнить целевую переменную;
- настроить ключевые гиперпараметры:
  - тип регуляризации,
  - коэффициент регуляризации,
  - балансировка классов;
- выполнить обучение модели на загруженных данных.

### Результаты обучения
После обучения отображаются основные визуализации и метрики:

- **ROC-кривая** — показывает способность модели различать классы;
- **Кривая Recall** — важна для задач, где пропуск положительного случая критичен;
- **Важность признаков** — демонстрирует, какие факторы оказывают наибольшее влияние на прогноз.

### Прогноз объекта
Раздел позволяет вручную вводить значения признаков для одного пациента или исследуемого случая.  
Модель выдаёт:

- предсказанный класс;
- вероятности классов;
- индивидуальную важность признаков для конкретного прогноза.

Также доступна возможность **экспортировать обученную модель**, чтобы использовать её вне приложения.

---

## CatBoost моделирование
Раздел CatBoost расширяет возможности пользователя и подходит для более сложных задач, включая как классификацию, так и регрессию.

### Основные возможности
В этом разделе можно:

- задать широкий набор гиперпараметров (глубина деревьев, скорость обучения, количество итераций и т. д.);
- обучить модель на тех же данных;
- просматривать важность признаков и метрики качества, характерные для градиентного бустинга.

CatBoost хорошо работает с разнородными медицинскими данными и может давать более высокий уровень точности.

### Прогноз объекта
Пользователь вводит значения признаков, после чего приложение предоставляет:

- предсказание модели;
- вероятности классов (если классификация);
- интерпретацию важности признаков для конкретного случая.

---

## Разъяснение результатов с ИИ
Этот раздел создаёт дополнительные пояснения и интерпретацию результатов, полученных в процессе моделирования. Система анализирует:

- структуру данных;
- целевую переменную;
- важность признаков;
- результаты моделей;
- показатели качества.

На основе анализа ИИ формирует понятные объяснения, которые помогают пользователю лучше ориентироваться в выводах модели, понимать, какие факторы играют ключевую роль, и какие возможны ограничения.

Пользователь может вводить текстовые запросы — например, уточнять смысл метрик, влияние отдельных признаков или интерпретацию конкретного прогноза — и получать развёрнутые пояснения.

---

Приложение предназначено для того, чтобы упростить работу с медицинскими данными и дать возможность строить модели, которые можно интерпретировать и применять в практических задачах.
